Introduction
************

Concept
*******

The parser takes one _rule_ as a input. Each rule has a _grammar_. A grammar is a series of rules.
It may as well contain _actions_ or _characters_.


About ``grap``
**************

``grap`` stands for **gra**\ mmar **p**\ arser. The library is heavily inspired on
`pest <https://pest.rs/>`_ - a PEG parser for the
`Rust programming language <https://www.rust-lang.org/>`_.
